UNK AI Chatbot (Capstone Project)

Tech Stack: Python, Flask, WebSockets, React, Tailwind CSS, MongoDB, Pinecone, OpenAI API

A full-stack conversational AI assistant designed to provide instant, accurate answers from the University of Nebraska at Kearney (UNK) website.

  • Document Ingestion: Crawls and parses data from the UNK site into a searchable format.
  • Vector Search: Uses Pinecone to enable semantic, context-aware query matching.
  • Real-Time Chat: Supports live group messaging with WebSocket integration.
  • Role-Based Access: Different UIs and permissions for staff vs. students.
  • Modern UI: Responsive frontend with markdown rendering and clean design.
  • Smart AI Layer: OpenAI GPT used to generate relevant, conversational responses.
